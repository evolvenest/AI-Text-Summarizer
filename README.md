# AI Text Summarizer

An AI-powered text summarization web application built using **Python**, **Google Gemini API**, and **Gradio**. This application allows users to input long-form text and generates concise summaries using Generative AI.

## 🚀 Features

- Summarizes long text into concise bullet points
- Interactive web interface built with Gradio
- Powered by Google's Gemini AI model
- Simple and user-friendly design
- Real-time text summarization

## 🛠️ Technologies Used

- Python
- Google Gemini API
- Google GenAI SDK
- Gradio
- Prompt Engineering
- Natural Language Processing (NLP)
- Generative AI

## 📌 How It Works

1. User enters the text to summarize.
2. The application sends the text to the Gemini API with a summarization prompt.
3. Gemini generates a concise summary.
4. The summarized output is displayed through the Gradio interface.

## ⚙️ Installation and Setup

### Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/AI-Text-Summarizer.git
cd AI-Text-Summarizer
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Configure Gemini API Key

Store your Gemini API key securely using environment variables or Google Colab Secrets.

### Run the Application

Open the notebook in Google Colab and execute all cells to launch the Gradio application.
